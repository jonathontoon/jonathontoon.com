Led the redesign of Rainforest QA's core test editor experience, making test automation more accessible through improved visual interfaces and streamlined workflows.
Key improvements included:
- Condensed and more versatile action blocks for easier test writing
- Enhanced run creation experience based on extensive customer feedback sessions
- Streamlined workflows that reduced complexity for non-technical users
- Visual editor improvements that made test automation more intuitive
The project involved extensive user research, customer feedback sessions, and iterative prototyping to understand which workflows made the most sense for core interfaces.
